# Python Scheduler

Python Scheduler is a lightweight tool for running Python tasks according to a timetable. Tasks can be configured to run either concurrently, allowing multiple tasks to execute at the same time, or sequentially, running one task after another.

The project is useful for automating recurring Python scripts and organizing scheduled jobs in one place.

## Features

- Schedule Python tasks using a timetable
- Run tasks concurrently or sequentially
- Automate recurring scripts and jobs

## Running the project

```bash
python app.py
```
